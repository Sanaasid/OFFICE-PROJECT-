# Backend Service

This backend is built with Node.js, Express, and MongoDB Atlas.

## Run locally

1. Install dependencies:
   ```bash
   npm install
   ```
2. Add `.env` with your MongoDB connection string.
3. Start the server:
   ```bash
   npm run dev
   ```

The API runs on `http://localhost:5000` and exposes `/api/seats` and `/api/bookings`.
