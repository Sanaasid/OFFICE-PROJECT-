# Smart Office Seat Booking System 🏢

A modern smart office seat booking and workspace management web application designed for hybrid office environments.

## 🌐 Live Demo
🔗 https://office-project-nine.vercel.app/

---

## 📌 Features

### Authentication
- User Login / Signup
- Secure Authentication System
- Role Based Access

### Dashboard
- Beautiful Responsive Dashboard UI
- Sidebar Navigation
- User Profile Dropdown
- Notification Panel

### Seat Booking Management
- Book Office Seats
- Release Reserved Seats
- Block Seats
- View Weekly Allocation

### Batch Rotation Logic
- Week 1:
  - Monday–Wednesday → Batch 1
  - Thursday–Friday → Batch 2

- Week 2:
  - Monday–Wednesday → Batch 2
  - Thursday–Friday → Batch 1

### Floating Seats
- Seats 41–50 Available for Everyone

### Holiday Management
- Holiday Calendar
- Booking Disabled on Holidays
